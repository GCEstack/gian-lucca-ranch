# Gian Lucca's Ranch

A sunny ranch page for Gian Lucca, built with love by Daddy.

## What's in this repo

This is a static-site deployment of **Gian Lucca's Ranch**, previously living inside the larger Runway project and deployed as a separate Vercel project (`gian-lucca-ranch`).

## Local development

```bash
npm install
npm run dev
```

Then open http://localhost:3000.

## Deploy

```bash
vercel --prod
```

## Asset sources

- Static build output: `02_DATA/kimi-agent-outputs/Kimi_Agent_Runway_App_Redesign/gian-lucca-site`
- Images: pulled from the existing Vercel deployment
- Videos/audio: copied from `03_ARCHIVE/gce-ai-runway/public/`
