# Story Scene Timing Guide

This guide maps each of the 10 scenes to the exact lines in your script so you can sync your voice recording to the visuals.

Each scene displays for **~18 seconds** with a gentle 0.5-second crossfade between scenes.

## Recording Tips

1. **Record in one continuous take** — the video builder expects one audio file per story
2. **Use the scene timings as a guide** — you don't need to be exact, but aim for roughly 18 seconds per scene
3. **Pause slightly between scenes** — the 0.5-second crossfade will smooth the transition
4. **Speak slowly and warmly** — this is bedtime story pace (~120 words per minute)
5. **After recording, run:** `scripts/build-videos.bat` (or the Python builder) to generate the MP4s

## Alternative: Python Video Builder

If you prefer Python, run:
```bash
python scripts/build-videos.py
```

This does the same thing as the batch file but works cross-platform.
